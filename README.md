# 📚 Library Management System using Core Java

A Java-based Library Management System developed independently during a **45-day Java Training Program conducted by Besant Technologies**. This project demonstrates the implementation of **Core Java**, **Object-Oriented Programming (OOP)**, and **JDBC** concepts to manage library operations efficiently.

---

## 🚀 Features

- 📖 Add new books
- 👤 Manage library members
- 🔄 Issue books
- 📥 Return books
- 🗄️ MySQL database connectivity using JDBC
- 💻 Console-based user interface

---

## 🛠️ Technologies Used

- Java
- Core Java
- JDBC
- MySQL
- Object-Oriented Programming (OOP)

---


---

## ⚙️ Prerequisites

- Java JDK 8 or above
- MySQL Server
- MySQL Connector/J
- Any Java IDE (Eclipse, IntelliJ IDEA, VS Code, NetBeans)

---

## ▶️ How to Run

1. Clone the repository.
2. Import the project into your preferred Java IDE.
3. Configure the MySQL database.
4. Add the MySQL JDBC Connector to the project.
5. Run `LibrarySystem.java`.
